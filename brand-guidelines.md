# Brand Guidelines

## Colors

| Role       | Hex       | Usage                          |
|------------|-----------|--------------------------------|
| Primary    | `#2D2D2D` | Headings, key UI elements      |
| Accent     | `#C49A6C` | CTAs, highlights, links        |
| Background | `#FAF8F5` | Page backgrounds               |
| Surface    | `#FFFFFF` | Cards, modals                  |
| Muted      | `#6B6B6B` | Secondary text                 |

## Typography

| Role        | Font Family        | Weight | Size (base) |
|-------------|--------------------|--------|-------------|
| Display     | Playfair Display   | 700    | 3rem        |
| Heading     | Playfair Display   | 600    | 2rem        |
| Body        | Inter              | 400    | 1rem        |
| Caption     | Inter              | 300    | 0.875rem    |

## Spacing Scale (Tailwind-compatible)

- `4` → 1rem
- `8` → 2rem
- `12` → 3rem
- `16` → 4rem

## Imagery Style
- Natural textures (wood grain, linen, stone)
- Warm, muted tones
- Clean product photography on light backgrounds

## Logo Usage
- Minimum clear space: equal to the cap-height of the wordmark on all sides
- Never distort or recolor the logo
